# Stardust Community Plugins

Community plugin collection for [Stardust](https://github.com/Tianzeyui/stardust).

## Plugin List

| Plugin | Description |
|--------|-------------|
| `_template` | Plugin development template — use as a starting point |
| `hello-world` | Getting started example plugin |
| `diary` | Diary plugin |
| `inspiration` | Inspiration capture plugin |

## Usage

Copy the desired plugin directory into your [Stardust](https://github.com/Tianzeyui/stardust) `plugins/` folder.

## Plugin Development

Use `_template` as a starting point — copy it and customize to begin building your own plugin.
